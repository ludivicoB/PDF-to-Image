# PDF to Image Converter 🖼️

A simple web application that converts PDFs to images using Flask (backend) and React (frontend).

## 🛠️ Technologies Used
- Flask (Python) - Backend API
- React.js - Frontend
- Axios - API Requests
- PyMuPDF (for PDF to Image conversion)
